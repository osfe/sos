Introduction¶

Le but de ce document est de fournir une documentation pour le logiciel SOS de Cliosoft à destination des administrateurs et des utilisateurs. Dans le présent document un administrateur est une personne pouvant être responsable d’un projet qui a la possibilité de procéder à des opérations de maitnenance.
